Stock Feed (api/vehicles)

Request Type: POST
Endpoint: https://services.codeweavers.net/api/vehicles

Full API Reference

Introduction

The Stock Feed endpoint is used to load vehicle stock into the Codeweavers system.
You are able to send an ExternalVehicleID, which is yours/the dealers ID for the vehicle. This ID will then be used in our other services to calculate finance for that vehicle.
Stock Vehicles have a TTL on our system of 7 days. Sending a new Stock Feed with the same vehicle details will reset the TTL.

Quick Start Guide

This quick start guide will get you started in integrating our core services for web providers. This guide will cover:

  • Stock Feed (api/vehicles)
    • Setting up a stock feed to import the dealers stock into the Codeweavers system for easy reference in the finance requests.
  • Calculate for Display (api/finance/calculatefordisplay)
    • Getting finance quotes on a number of vehicles.
  • Payment Search (api/vehicles/search)
    • Search for vehicles and finance based on the finance parameters provided.
  • Finance Calculator Plugin
    • Integrating the finance calculator plugin for a single vehicle. Usually used on a vehicle details page.
  • Standalone Apply For Finance Form
    • Integrating the Standalone Apply Form for customers to make an application without a vehicle or quote. Usually used on a Finance page.

API Specification

  • Platform: HTTP
  • API Type: RESTful
  • Error Response codes: HTTP
  • JSON Availability: Yes
  • Libraries: HTTP / REST

Cross-Origin Resource Sharing

This API features Cross-Origin Resource Sharing (CORS) implemented in compliance with W3C spec, which allows cross-domain communication from the browser. All responses have a wildcard same-origin which makes them completely public and accessible to everyone.


Errors

Error Code Meaning
400 Bad Request -- Your request is incorrect
401 Unauthorised -- Your credentials are incorrect
403 Forbidden -- You do not have permission to use the service. Error information may indicate why.
404 Not Found
429 Too Many Requests -- You're calling the service too often
500 Internal Server Error -- We had a problem with our servers. Please try again later.
503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.

Calculate for Display

Request Type: POST
Endpoint: https://services.codeweavers.net/api/finance/calculatefordisplay

Full API Reference

Introduction

Calculate for Display is used to get finance for one or multiple vehicles, which is returned in a JSON response. This is mostly used on a listings page to display the finance. You can pass in either Stock Id's (if using a stock feed), or vehicle details.

Example use of Calculate for Display

Integration

You can make a request to our service using either a Stock Id (if using Stock Feed) or vehicle details. Below are examples of the minimum amount of information required to return a quote. The response returned includes the full quote, along with "QuoteActions", which are links for customers to continue their finance journey.

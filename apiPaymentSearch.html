Payment Search

Request Type: POST
Endpoint: https://services.codeweavers.net/api/vehicles/search

Full API Reference

Introduction

Payment Search is used to filter vehicles on a listings page by selected finance parameters. For example, the customer could select the parameters for the minimum and maximum monthly payments, finance product, term length, annual mileage and deposit. Passing through the Stock IDs and these parameters into this service will return quotes for all the vehicles that fit within these parameters.

Integration

The Payment Search service requires use of the Stock Feed to function.
